# BOLS
code of BOLS algorithm

## Usage
Using ./build.sh to build the solver 
To run the solver, using ./solver <demand_file_path> <supply_file_path> <time_limit> <parameter_1> <parameter_2>

<parameter_1> <parameter_2> is $t_s$ in the paper
